# Provider Rails

## Supported Rails
- MTN MoMo
- Vodafone Cash
- AirtelTigo Money
- MoveGH Wallet

## Adapter Contract
- initiatePayment
- verifyPayment
- webhookHandler
- refund
- payout

## Security Requirements
- Signed requests
- Nonce and timestamp
- Idempotency headers
- Correlation IDs
- Webhook verification and dedupe
